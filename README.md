## Install Yeoman, Grunt and Bower
	$ npm install -g yo grunt-cli bower

## Setting Up AngularJS using generator-angular template
	$ npm install -g generator-angular
	$ yo angular 

## Run application
	$ npm install
	$ bower install
	$ grunt server
